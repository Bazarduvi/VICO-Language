📜 POLÍTICA DE PRIVACIDAD

VICO Lengua — Aplicación de aprendizaje de idiomas.

Última actualización: Marzo 2026

Todos los datos de usuario (notas, tarjetas, historial) se almacenan localmente en tu dispositivo mediante localStorage. No enviamos datos a ningún servidor propio.

  • Las API Keys (Gemini/Groq) se guardan solo en tu dispositivo y se envían directamente a los servidores de Google/Groq para las solicitudes de IA.
  • No utilizamos cookies de terceros ni rastreadores.
  • No compartimos ningún dato personal con terceros.
  • Puedes exportar o borrar todos tus datos en cualquier momento desde Perfil → Backup de Datos.

Aviso de uso de IA: Las traducciones y generaciones con IA se procesan por Google Gemini o Groq AI según la API Key configurada. Consulta sus políticas de privacidad respectivas.

⚖️ TÉRMINOS DE USO

Al usar VICO Lengua aceptas los siguientes términos:

  • Esta aplicación se proporciona "tal cual" sin garantías.
  • Eres responsable del uso de tus propias API Keys.
  • El contenido educativo es solo con fines de aprendizaje.
  • No reproducir ni distribuir el contenido sin autorización.
  • Los créditos gratuitos son limitados y no reembolsables.

Versión PRO: La activación PRO mediante llave maestra te permite funciones avanzadas de IA sin límite de créditos, siempre que tengas API Keys propias configuradas.
